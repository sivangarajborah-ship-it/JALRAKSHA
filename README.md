## Problem Statement
Floods cause major loss of life and property due to lack of timely alerts and awareness.
There is a need for a simple system that can inform people early and help them stay safe.

## Solution
JalRaksha is an Android application that provides flood-related alerts and awareness information
in a simple and user-friendly way. It is designed as an MVP for hackathon use.

## Features
- Flood alert information (demo)
- Weather-related updates
- Simple and clean user interface
- Easy to install and use
